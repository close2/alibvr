template<typename T, typename T2, T(*)(T2) F>
struct tasks::TaskWrapper< T, T2, F >

A minimalistic wrapper for tasks.

For variadic template lists functions would only be possible, if they all had the same type. This simple wrapper "converts" tasks/functions to classes.
