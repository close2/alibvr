Detailed Description

"The measured voltage has a linear relationship to the temperature [...]. The voltage sensitivity is approximately 1 mV/°C and the accuracy of the temperature measurement is +/- 10°C." (Atmega docs)

Typical values:

  • -45°C: 242mV
  • +25°C: 314mV
  • +85°C: 380mV These values are typical values.

To achieve more accuracy calibrate the temperature sensor by storing a calibration value in the eeprom. See the atmel docs for more information.
