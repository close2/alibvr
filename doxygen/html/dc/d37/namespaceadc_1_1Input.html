Detailed Description

In addition to the ADC-pins the adc subsystem can also use an internal temperature sensor, an internal 1.1V reference voltage and ground as input for a conversion.

You may use Input::Temperature, Input::V1_1 and Input::Gnd as if they were ADC-pins.

The required static fields port and bit are implemented and have special values, so that the code of this library knows when to switch to those inputs.
